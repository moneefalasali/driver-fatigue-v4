# Driver Fatigue V4

A Flask + Socket.IO web application for monitoring driver fatigue using AI-based analysis and sensor data.

## Features
- Driver fatigue monitoring dashboard
- Real-time sensor and camera integration
- JWT authentication
- Admin dashboard
- PostgreSQL-ready configuration for Render deployment

## Local Development
1. Create and activate a virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python run.py`

## Deployment
This project is configured for deployment on Render with PostgreSQL.
